# 🌿 Manga Collector | Boticaria Selection

**Manga Collector** es una plataforma web minimalista y elegante diseñada para la gestión de pedidos de manga en Panamá. Inspirado en una estética "Boticaria" (limpia, natural y con acentos botánicos), el sitio permite a los estudiantes de la Universidad de Panamá (UP) y coleccionistas locales explorar un catálogo y coordinar sus compras directamente vía WhatsApp.

---

## ✨ Características Principales

* **Catálogo Interactivo:** Visualización de tomos con tarjetas animadas y efectos hover.
* **Sistema de Carrito Dinámico:** Gestión de pedidos en tiempo real con cálculo automático de totales.
* **Barra de Promociones Inteligente:** Avisos dinámicos en el header que rotan mensajes de estado y contacto.
* **Logística de Entrega Personalizada:** Opciones específicas para retiros en MedMarket (UP), Biblioteca, Albrook o envíos nacionales.
* **Integración con WhatsApp:** Generación de mensajes automáticos estructurados para finalizar la compra.
* **Diseño "Kawaii-Boticaria":** Paleta de colores basada en verdes profundos (`#0F5F5F`), turquesas (`#1B8A8F`) y amarillos cálidos (`#FFD54A`).

## 🛠️ Tecnologías Utilizadas

* **HTML5 / CSS3:** Estructura semántica y animaciones personalizadas.
* **Tailwind CSS:** Framework principal para un diseño responsivo y moderno.
* **JavaScript (Vanilla):** Lógica del carrito, rotación de promociones y cálculos de cupones.
* **Google Fonts:** Uso de *Playfair Display* para títulos clásicos y *Outfit* para lectura clara.

## 📁 Estructura del Proyecto

* `index.html`: Página principal y estructura del sitio.
* `styles.css`: Estilos personalizados (scrollbars, sombras y animaciones).
* `script.js`: Lógica del catálogo, carrito y barra promocional.
* `descuentos.js`: Sistema de validación de cupones.
* `img/`: Directorio de recursos visuales y portadas de manga.
