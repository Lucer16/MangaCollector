# Manga Collection - Catálogo Boticaria 🌿

Manga Collection es una aplicación web responsiva diseñada para la gestión y visualización de un catálogo de mangas. El proyecto se enfoca en ofrecer una experiencia de usuario fluida, visualmente atractiva y optimizada para dispositivos móviles.

## ✨ Característicasprincipales

- **Catálogo Dinámico:** Renderizado automático de productos desde un archivo JSON.
- **Buscador Inteligente:** Filtrado en tiempo real por título y código de barras (EAN).
- **Sistema de Filtros:** Clasificación por editorial y ordenamiento por precio.
- **Carrito de Compras:** Gestión de productos seleccionados con cálculo de total automático.
- **Integración con WhatsApp:** Generación de pedidos estructurados que incluyen detalles de envío y productos.
- **Diseño "Kawaii-Elegant":** Interfaz construida con Tailwind CSS, utilizando una paleta de colores personalizada (teal, amarillo y rosa suave).

## 🛠️ Tecnologías utilizadas

- **HTML5 & CSS3** (Animaciones personalizadas y Google Fonts).
- **Tailwind CSS** (Framework de estilos).
- **JavaScript (ES6+)** (Manipulación del DOM, lógica de filtros y persistencia de datos).
- **JSON** (Base de datos local para productos).

## 📁 Estructura del Proyecto

- `index.html`: Estructura principal y contenedores.
- `mangas.json`: Base de datos con títulos, precios, editoriales y códigos EAN.
- `/img`: Directorio para las portadas de los mangas (Relación de aspecto recomendada 3:4).
